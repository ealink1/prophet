## 关键业务指标分析

### 收入数据

- **今日收入 ¥317.40** / **累计收入 ¥979.80**
- 今日收入占累计收入的 **32.4%**，说明今天是收入较高的一天
- 平均每日收入约 ¥979.80 / 天数

### 用户数据

- **累计用户 269**，今日活跃 11（活跃率约 4.1%）
- 历史记录 124，平均每个用户约 0.46 条记录
- 今日解锁 5，说明有 5 个用户解锁了付费内容

### 订单数据

- **累计订单 81**，已支付 70（支付成功率 **86.4%**）
- 待支付 10，待审核 0
- 待支付订单较多，可能需要关注转化

## 刷新功能

- 右上角"🔄 刷新"按钮
- 点击后重新加载所有统计数据
- 使用 `font-display` 金色字体

## UI 细节

- 标题"数据看板"使用 `font-display` 金色字体（志莽行体）
- 卡片使用深色背景 + 浅色边框
- 数值使用大号粗体白色
- 标签使用小号灰色
- 整体风格与前台保持一致的玄学暗色调
