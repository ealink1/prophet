# 后台管理系统分析

## 系统概览

菩提苑管理后台是一个独立的 Web 管理系统，用于管理前台应用的运营数据、订单、用户、商品等内容。

## 界面风格

- **深色主题**：与前台保持一致的玄学风格深色背景
- **左侧导航栏**：固定左侧，深色底，带 logo 和功能菜单
- **主内容区**：右侧，卡片式布局
- **配色方案**：金色（主色）+ 深棕背景，与前台视觉统一

### 导航菜单功能说明

| 菜单项 | 功能 |
|--------|------|
| 数据看板 | 核心数据概览（收入、用户、订单） |
| 订单审核 | 订单列表管理，确认到账/拒绝 |
| 用户列表 | 用户信息管理 |
| 商品定价 | 前台付费产品定价配置 |
| 收款码 | 收款二维码管理 |
| 禅修音乐 | 音乐曲目管理 |
| 系统配置 | 全局配置参数 |
| 登录安全 | 后台登录安全设置 |
| 审计日志 | 操作日志记录 |
| 修改密码 | 管理员密码修改 |

### 角色

- 底部显示"登录身份：超管"（超级管理员）
- 有退出登录按钮
