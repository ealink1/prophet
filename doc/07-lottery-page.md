`/lottery/` → 关帝灵签页面

## 师父角色系统

三个 AI 导师角色，各有不同风格：

| 师父 | Emoji | 角色 | 风格 | 适合场景 |
|------|-------|------|------|----------|
| 慧明长老 | 🧘 | 古寺住持 | 庄重持重，引经据典 | 深度解读、古籍出处 |
| 明心师父 | 🙏 | 尼众法师 | 慈悲温柔，劝人向善 | 家庭、感情、祈福 |
| 玄真道长 | ☯️ | 山中道人 | 直爽通透，说大白话 | 急性子、直接解读 |

## 交互流程

1. 阅读标题和说明
2. 选择一位师父（卡片式选择器）
3. 输入想要问的问题
4. 点击 "默念所求 · 求一支签"
5. 系统返回签文 + 师父开示

## 签文数据结构（推测）

```javascript
{
  code: "第X签",
  title: "签题",
  level: "上上" | "上吉" | "中吉" | "中平" | "下下",
  poem: "签诗内容...",
  analysis: "签文解析...",
  master_reading: "师父开示（根据选择的角色生成）...",
  lucky: { ... },
  advice: "..."
}
```

## 免费配额

- 登录后查看每日免费配额
- 未登录用户可能有次数限制
- 免费额度用完后引导付费

## UI 细节

- 师父选择器：卡片式，可能带边框高亮
- 每个师父卡片包含：emoji 图标、名称、角色、风格描述
- 问题输入区：文本框，placeholder 引导
- 求签按钮：金色 CTA，可能带动画效果
