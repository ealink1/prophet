# 菩提苑 · 网站总览

## 网站定位

菩提苑是一个**中国传统文化/命理/祈福类 PWA 应用**，面向有祈福、命理、灵签等需求的用户。口号："心诚则灵。为家人点一盏祈福灯，求一支关帝灵签，看一卦命理八字。一念慈悲，福报自来。"

## 核心功能模块

| 模块 | 路径 | 核心功能 |
|------|------|----------|
| 为家人祈福 | `/qifu/` | 点功德灯、供奉时长、功德灯墙展示 |
| 今日黄历 | `/almanac/` | 每日宜忌、吉凶、时辰、七日历 |
| 求灵签 | `/lottery/` | 关帝灵签、选择师父开示 |
| 八字精批 | `/bazi/` | 生辰八字排盘、命理分析 |
| 周公解梦 | `/dream/` | 梦境描述 → 解梦、按类查梦 |
| 看手相 | `/palmistry/` | 手相图解读 |
| 宝宝起名 | `/naming/` | 五行/音韵/笔画评分起名 |
| 六爻占卜 | `/divination/` | 六爻卦象分析 |
| 静心禅坐 | `/meditation/` | 禅修音乐播放、曲目列表 |

## 商业模式

- **祈福灯供奉**：¥6.6/盏（功德灯）
- **付费报告**：单次付费产品（流年运势 ¥9.9、八字精批深度版 ¥19.9、手相图解读 ¥19.9、合婚 ¥29.9、宝宝起名 VIP ¥49.9、公司起名 ¥99.9）
- **分享返佣**：用户分享专属链接，朋友下单后获得福报金奖励
- **吉祥号**：邀请码/吉祥号系统

## 目标平台

- 移动端优先（PWA 全屏 App 体验）
- 微信/抖音内嵌浏览器适配
- 桌面端兼容（md breakpoint）
